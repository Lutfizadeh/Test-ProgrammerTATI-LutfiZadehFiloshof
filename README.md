- Run ```composer install``` for install dependecies.
- Run ```php artisan shield:super-admin``` for create Super Admin.
- Access ```http://127.0.0.1/admin``` for login.

- Features:
1. Create log
2. Verification log
3. Create user
4. Show stats
5. Show line chart

- Techstack: Laravel Filament
